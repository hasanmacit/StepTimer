# StepTimer

Oyunun normal timer sistemi yerine microsoftun timer sistemini kullanır. Sisteminizin desteklediği maximum fpsi alırsınız ve ekrandaki yenileme hızları ona göre şekillenir. Fps sınırı yoktur.

English:
The game uses Microsoft's timer system instead of the standard timer system. You get the maximum FPS your system supports, and the screen refresh rates adjust accordingly. There is no FPS limit.
